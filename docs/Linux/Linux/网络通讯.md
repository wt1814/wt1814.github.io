&emsp; 几个比较好用的小工具，iftop和nethogs，用于排查linux机器的网络问题。  

### iftop  
&emsp; iftop用于网卡机器级别的流量监控，可以实时显示当前机器和其他主机之间的网络流量。  

### nethogs
&emsp; nethogs用于监控统计每个进程的带宽使用。
