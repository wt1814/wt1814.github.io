&emsp; elasticsearch有六种搜索方式：  
1. query string search。  
2. query DSL，DSL(Domain Specific Language特定领域语言)以JSON请求体的形式出现。  
3. query filter，对数据进行过滤。  
4. full-text search，全文检索。  
5. phrase search，短语搜索。  
6. highlight search，高亮搜索。
