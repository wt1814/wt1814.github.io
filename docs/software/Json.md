## Json文件注释
&emsp; 如果要严格根据json规范，是不建议在json文件上添加注释符号的，我们可以使用重复定义的方法来添加注释。  
&emsp; 比如一个键定义二次，第一次的值放上注释内容，第二次的值才是真正的值。json文件解析后，是使用最后定义的值的。
